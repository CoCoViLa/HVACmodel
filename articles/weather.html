Weather

Weather is the component, which is required for setting up parameters for the air: temperature, heat capacity and density. To set up the temperature this component is combined with scheduler of any type. Heat capacity and density are defined by user as properties: airDensity (default value=1.2250 kg/m3) and c (default value=1005.4 J/kgâˆ’K). As output this component has triple alias airProperties = (temperature, c, airDensity).

The Weather can be combined with some of schedulers to define temperature. Scheduler24hSin proposes the most natural way for controlling the outdoor temperature during 24h.
