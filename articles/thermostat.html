Component: Thermostat

Thermostat is the control component, that sends signal to AirConditioner to heat up or cool down the air. The decision is done by comparing the required temperature and current temperature in the room.

There are only two properties of this component: name and tolerance (default value 0.2). The tolerance shows how big can be difference between required and current temperatures.

Thermostat compares two temperatures and produces signal:

  • "-1" if cooling is required;
  • "0" if temperature is OK;
  • "1" if heating is required;
